# Design Placeholders

This directory is intended to store placeholder assets (images, graphics, icons) used during development before final assets are provided by the design team.

- Add temporary image assets here.
- Reference them in mock data or directly in UI code during testing.
- Replace them with final assets before product launch.

Currently using `cached_network_image` with dummy URLs (`picsum.photos`) in most places, but file-based placeholders can be placed here.
